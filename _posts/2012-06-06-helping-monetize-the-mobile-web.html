Game developers building on the mobile web have been able to use Facebook Payments to monetize their apps for some time.A full list of supported countries can be found here.However, this flow for mobile payments can be long, with a total of up to seven steps, including SMS verification.At Mobile World Congress this year we announced an improved mobile payments flow for mobile web apps.Today we are starting to roll it out.
