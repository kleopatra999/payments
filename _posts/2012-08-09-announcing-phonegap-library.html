We are proud to announce the release of the Urban Airship PhoneGap library, which is available as of August 8, 2012.We are shipping the library with support for iOS and Android.This release wraps our existing libraries for both platforms, allowing users to easily embed Urban Airship in their applications.Wrapping our client libraries means that you can include push messaging, location history, and results reporting in your application using just a few lines of code.To make it easier for you to use the library, the same JavaScript API is available for both iOS and Android, so you can use the same JavaScript code on both platforms, with little or no changes.
